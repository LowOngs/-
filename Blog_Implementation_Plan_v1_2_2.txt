■ PC2 (데스크톱)

[헤더]
- 2층 구조. 
- 중앙: 로고(토끼) + 블로그명 + 소개글. 
- 소개글 끝에 구독버튼(운영자 로그인시 관리버튼). 
- 우측 끝: 구글 번역 위젯(드롭 3선택: 구글, 파파고, 딥엘).

[본문]
- 독립 스크롤 가능, 연초록(#CCE2D8) 1px 테두리 + 종이 질감 배경(#FAFAF8). 
- 좌우 네비게이션 버튼: 중앙 2/3 위치, << / >> 모양, 초록 글로우 방향성 애니. 
- 마우스 포인터 접근·스크롤 시 노출, 1초 후 사라짐 규칙. 
- 글번호 자동 삽입 (#123), 검색창에서 검색 시 해당 글 이동.

[사이드바]
- 공지사항 마키, 카테고리 아코디언(토끼 아이콘, 후광효과 0.6s), 광고 슬롯 2개(텍스트+디스플레이), 특가 안내 3라인(Deal API 연동), 방문자 집계(일간/월간/연간/누적, 색상규칙). 
- 아코디언 하부 메뉴 공간 상시 예약 → 밀림 없음.

[푸터]
- 1층: 파트너 12아이콘 1열(가로 스크롤), Deal 활성시 Deal 마크 표시(API 기준). 
- 2층: 제휴 고지문구(disclosure_text). 고정 출력. 
- 3층: 카피라이트(copyright_text, 좌)/카카오문구(kakao_notice_text, 우).

■ PC1 (모바일)

[헤더]
- 1층: 중앙(로고+블로그명), 우측 끝 햄버거 메뉴. 
- 2층: 좌(◀ 이전글), 중앙(구독/관리 버튼), 우(▶ 다음글). 
- 헤더 sticky. 
- 검색창 placeholder: “글검색: 예) #123”.

[본문]
- 단일 컬럼, 패딩 16px, 종이 질감 배경(#FAFAF8). 
- 네비게이션은 헤더 버튼만. 
- 글번호 #123 상단 표시, 검색 트리거 가능.

[사이드바 (Drawer)]
- 진입 시 기본 오픈(광고 노출), 이후 사용자 선택 기억. 
- 1층: 번역 위젯 드롭(구글/파파고/딥엘). 
- 3~4층: 공지사항. 
- 6층: 카테고리 아코디언(주황/금색 후광). 
- 광고 슬롯(텍스트+디스플레이), 특가 안내 3라인, 방문자 집계(일간/월간, 색규칙). 
- 아코디언 공간 예약 → 밀림 없음.

[푸터]
- 1층: 파트너 12아이콘(PC1 전용 PNG, 1:1 비율). 
- Deal 마크: “D” 아이콘(안쪽 파란/바깥 초록 2중 라인), 흔들림 애니(±10°, 0.3s×2, 2.7s 정지, 3s 반복). 
- 2층: 제휴 고지문구(폰트 작게, 2줄 고정 출력). 
- 카피라이트/카카오 문구 없음.

────────────────────────────────────────
■ 댓글 번역 오브 (PC1/PC2 공통)
- 댓글 상단 “번역보기” 클릭 시 오브(원형) 등장. 
- 오브 옆 막대 → 깃발 펼침 → 구글/파파고/딥엘 선택 버튼 표시. 
- 선택 후: 같은 깃발 안에 번역문 출력, 하단에 [번역기 재선택] 버튼. 
- 닫기: 오브 상단 X(전체 종료) / 번역문 X(번역문만 닫기 → 깃발 닫힘, 막대+오브 유지). 
- 오브 드래그 이동/위치 기억. 
- 번역문만 닫았다가 다시 선택 시 깃발 재출현.

■ 공통 기술/규칙
- Zero-Content Safe: 글이 0개여도 뼈대·푸터·사이드바 출력 유지.

- 다국어: 번역 위젯(구글/파파고/딥엘) 공통 적용. 
- 접근성: aria-label, aria-expanded, 포커스 링, 최소 터치 타깃 44px. 
- 성능: lazy-load 이미지, 지연 스크립트, transform 애니 사용.
